# Notebooks

This folder contains the implementation notebooks used in the Arabic Fake News Detection project.

## Contents

### arabic_fake_news.ipynb
Implementation and evaluation of traditional Machine Learning and Deep Learning models for Arabic fake news classification, including:
- Support Vector Machine (SVM)
- Naive Bayes
- LSTM

### arabert.ipynb
Implementation of the AraBERT transformer model for Arabic fake news detection and performance evaluation.

### hybrid_marbert_lstm_reconstruction.md
Documentation of the Hybrid MarBERT + LSTM architecture and project results. The original notebook is currently being reconstructed from the project report and presentation materials.

## Technologies Used

- Python
- PyTorch
- Hugging Face Transformers
- Scikit-learn
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Objective

To compare traditional machine learning, deep learning, and transformer-based approaches for multiclass Arabic fake news classification.
